# Agents

Firstly enter into the Agents folder
```
cd examples.composabl.io\agents
```

Install poetry dependencies:
```
poetry install
```

Then you can use any of these commands to execute an agent using poertry
```
poetry run cstr
poetry run airplane
poetry run starship
poetry run boiler
```
